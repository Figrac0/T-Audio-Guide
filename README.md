T-Guide - SPA для поиска мест рядом, просмотра готовых аудиоэкскурсий, сборки личного маршрута и работы с профилем пользователя.

## Запуск

```bash
npm install
npm run dev
```

Проверки перед сборкой:

```bash
npm run typecheck
npm run lint
npm run build
```

## Основные страницы

| Страница          | URL                | Назначение                                                                            |
| ----------------- | ------------------ | ------------------------------------------------------------------------------------- |
| Главная           | `/`                | Карта рядом с пользователем, фильтры мест, готовые экскурсии, быстрый просмотр точки. |
| Маршруты          | `/excursions`      | Сборка пользовательского маршрута из точек на карте и каталог готовых маршрутов.      |
| Страница маршрута | `/excursion/:slug` | Описание конкретной экскурсии, точки маршрута, старт прохождения.                     |
| Профиль           | `/profile`         | Данные пользователя, сохраненные прогулки, свои маршруты, история прохождений.        |
| Авторизация       | `/sign-in`         | Вход и регистрация пользователя.                                                      |

## Главная страница

Основные файлы:

| Файл                                         | За что отвечает                                                        |
| -------------------------------------------- | ---------------------------------------------------------------------- |
| `src/pages/home/ui/HomePage.tsx`             | UI главной страницы, bottom sheet, фильтры, карточки мест и маршрутов. |
| `src/pages/home/ui/HomePage.css`             | Адаптив, стили шторки, карточек, футера и декоративных элементов.      |
| `src/features/route-map/ui/DiscoveryMap.tsx` | Leaflet-карта главной страницы, маркеры, popup точки, радиус поиска.   |
| `src/features/route-map/ui/DiscoveryMap.css` | Стили карты, popup, контролов и состояния загрузки.                    |

Функционал:

- показывает карту на весь экран;
- отображает места рядом с пользователем;
- фильтрует места по категориям;
- открывает popup точки на карте;
- показывает подробности точки в шторке;
- строит маршрут до выбранной точки;
- показывает готовые экскурсии в нижней шторке;
- для неавторизованного пользователя оставляет доступной главную страницу, но защищает переходы к закрытым сценариям.

## Страница маршрутов

Основные файлы:

| Файл                                                   | За что отвечает                                                                        |
| ------------------------------------------------------ | -------------------------------------------------------------------------------------- |
| `src/pages/excursions/ui/ExcursionsPage.tsx`           | UI страницы маршрутов, bottom sheet, блок "Мой маршрут", каталог и подробности точки.  |
| `src/pages/excursions/ui/ExcursionsPage.css`           | Стили шторки, каталога, карточек, detail panel и адаптива.                             |
| `src/pages/excursions/ui/RouteBuilderMap.tsx`          | Карта сборки маршрута, popup точки, добавление точки, отображение выбранного маршрута. |
| `src/pages/excursions/ui/RouteBuilderMap.css`          | Стили popup, маркеров, кнопок карты и route-layer.                                     |
| `src/pages/excursions/model/useExcursionsPageState.ts` | Основная логика страницы: фильтры, выбранные точки, сохранение, detail state.          |

Функционал:

- доступна только авторизованному пользователю;
- открывает popup точки прямо на карте;
- добавляет точку в пользовательский маршрут;
- поддерживает до 6 точек в маршруте;
- показывает порядок точек на карте числами;
- строит маршрут от пользователя к первой точке и дальше по выбранной последовательности;
- дает сохранить или сбросить собранный маршрут;
- показывает подробное описание точки в шторке;
- в подробном описании точки есть кнопки "Добавить в свой маршрут" и "Закрыть";
- фильтрует готовые маршруты по категории и времени.

## Страница отдельной экскурсии

Основные файлы:

| Файл                                             | За что отвечает                                                                         |
| ------------------------------------------------ | --------------------------------------------------------------------------------------- |
| `src/pages/excursion/ui/ExcursionPage.tsx`       | Верхний блок маршрута, точки маршрута, старт прохождения, навигация между точками.      |
| `src/pages/excursion/ui/ExcursionPage.css`       | Адаптив страницы маршрута, карточки метрик, сетка точек, футер и декоративные элементы. |
| `src/entities/excursion/model/useRouteBySlug.ts` | Получение маршрута по slug из mock-каталога.                                            |
| `src/entities/place/ui/RouteStopList.tsx`        | Список точек маршрута.                                                                  |

Функционал:

- показывает описание маршрута;
- выводит метрики маршрута: время, точки, длина, формат;
- показывает точки маршрута с изображениями и доступностью аудиосопровождения;
- поддерживает старт маршрута и переход по точкам;
- обновляет локальную историю прохождения маршрута.

## Профиль

Основные файлы:

| Файл                                                    | За что отвечает                                                      |
| ------------------------------------------------------- | -------------------------------------------------------------------- |

- доступен только авторизованному пользователю;
- позволяет менять имя, почту и язык аудиогида;

- показывает сохраненные прогулки;
- показывает пользовательские маршруты;
- хранит историю последних маршрутов в `localStorage`;
- отображает прогресс прохождения маршрутов.

## Данные и backend-ready слой

- Готовые маршруты и места сейчас работают на mock-данных.
- Сохраненные и пользовательские маршруты хранятся во frontend-state и `localStorage`.
- История последних маршрутов хранится в `localStorage` под ключом `lastRoutes`.
- API-слой вынесен в `src/shared/api`.
- Контракты пользователя, профиля, авторизации и токенов описаны в `src/shared/api/contracts.ts`.
